What is MangoHud?

MangoHud is an open-source overlay for monitoring and displaying real-time performance metrics (FPS, frame timings, GPU, CPU, etc.) in Vulkan and OpenGL games on Linux. It's widely used for gaming performance analysis and system benchmarking.

Features

  • Monitor FPS, frame times, and system performance metrics.
  • Supports Vulkan and OpenGL applications.
  • Highly customizable through configuration files.
  • Open-source and lightweight, minimal performance overhead.
